AI & Research Projects

Adaptive Reward Clustering and Sampling (ARCS)

Developed an adaptive reward-based experience clustering and sampling method to improve replay efficiency, convergence stability, and learning performance in deep reinforcement learning systems.

Technologies: Python, PyTorch, Deep Reinforcement Learning, Experience Replay

DRL-Based Resource Allocation in Vehicular Edge Computing

Designed reinforcement learning models for latency-aware, energy-efficient, and constraint-aware task offloading in vehicular edge computing environments. The framework optimizes task completion, communication resources, and system reliability.

Technologies: DRL, DQN, DDPG, Vehicular Edge Computing, Wireless Resource Allocation

Multi-Agent Reinforcement Learning for Distributed Control

Developed multi-agent reinforcement learning frameworks for coordinated decision-making under uncertainty in distributed cyber-physical systems, with emphasis on stability, scalability, and reliable system operation.

Technologies: MARL, CTDE, MADDPG, COMA, Distributed Systems

Bilevel Leader–Follower Optimization Framework

Designed a hierarchical leader–follower reinforcement learning framework for constrained resource allocation, combining centralized coordination with distributed agent-level decision-making.

Technologies: Stackelberg Game, CTDE, Multi-Agent DRL, Resource Allocation

LSTM-MHA-Assisted DRL Caching for Vehicular Edge Networks

Contributed to a proactive edge caching framework using LSTM, multi-head attention, and DDQN to improve cache hit ratio, latency, and quality of experience in vehicular edge networks.

Technologies: LSTM, Multi-Head Attention, DDQN, Edge Caching, Cabspotting Dataset

Traffic Sign Classification using CNNs

Developed and optimized a CNN-based traffic sign classification system using the GTSRB dataset, comparing learning rates, batch sizes, network structures, and optimizers to analyze convergence and classification performance.

Technologies: CNN, TensorFlow/Keras, Python, Computer Vision, GTSRB Dataset

Industrial Automation Projects

Schneider ETAP-Based Energy Management System

Implemented a Schneider ETAP-based EMS-SCADA system using Modbus TCP/IP communication to integrate field energy meters, PLCs, and plant-level monitoring systems for real-time energy monitoring, load analysis, and energy-efficiency improvement.

Technologies: Schneider ETAP, SCADA, Modbus TCP/IP, PLC, Energy Monitoring

33 kV/11 kV/0.4 kV, 4 MVA Substation Installation and Commissioning

Led installation and commissioning of industrial substations involving power distribution, protection coordination, transformer integration, and electrical infrastructure setup for large-scale manufacturing systems.

Technologies: Power Distribution, Protection Coordination, Industrial Electrical Systems

Technologies: KRONES, PLC, VFD, SCADA, Industrial Ethernet, High-Speed Automation
